## js 数据类型
    简单数据类型
        - number 不分长整 短整 浮点 都是数值类型
        - string
        - boolean
        - null 空 定义了，但值为空
            值为空
        - undefine 声明了变量（只声明），但未赋值
            内存空间（undifine 有地址，但是地址长度不定 ） 弱类型语言

    复杂数据类型
        - object
        除了简单数据类型 都是对象
        arry[] 可遍历的对象
        object{} 对象字面量
        function 可执行的对象

        - 自学作业 typeof
            可以获得自变量类型，但是不准确
